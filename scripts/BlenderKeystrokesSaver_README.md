# About
This script is intended to save keystrokes and vertices and other objects to file so it is easier to feed AI models to teach them how a human approaches modeling of 3D objects.

# Status
working POC, let's see the results.

# Jak sprawdzić czy działa?
w folderze %AppData%\Blender Foundation\Blender\5.1\
utwórz folder scripts
w którym utwórz folder startup
Wrzuć plik BlenderKeystrokesSaver.py do %AppData%\Roaming\Blender Foundation\Blender\5.1\scripts\startup

Uruchom Blendera.

Przejdź w Edit Mode, zaznacz kilka kratek, użyj G (Move) lub E (Extrude).
